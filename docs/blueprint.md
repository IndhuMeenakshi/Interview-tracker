# **App Name**: TrackStar Interviews

## Core Features:

- Company Entry: Allow users to enter the name of companies they've interviewed with.
- Stage Tracking: Enable users to log each interview stage (application, phone screen, technical interview, etc.)
- Offer Letter Upload: Let users upload the offer letter with a direct URL. LLM tool scans the link, retrieves key info (compensation, start date) and populates fields.
- Selection Status: A 'Selected' status indicator for keeping track of positive outcomes.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to evoke trust and reliability, reflecting the app's serious purpose.
- Background color: Light gray (#F0F4F8), a desaturated hue of the primary, for a clean and professional backdrop.
- Accent color: Soft green (#81C784), an analogous color to the primary, used to highlight positive outcomes like 'Offer Received'.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable interface.
- Simple, professional icons to represent each stage of the interview process.
- Clean, structured layout with clear visual hierarchy to easily track the interview stages and outcomes.
- Subtle transitions and animations to provide feedback on user interactions.
